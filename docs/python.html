You've enabled Python, now what?

The data.world python package makes it easy to download datasets, run queries, and upload data all using data.world's REST API.

Install and configure data.world using pip.

You'll use Python's package manager, pip, to install the data.world package:

  1. Using the command line, install the data.world package from PyPI.
    2. $ pip install datadotworld[pandas]
  2. Configure the package via the command line.
    3. $ dw configure
    Note: You will need to have your data.world API token, which you can get from data.world's advanced settings.

Import data.world and begin using python.

Using a python shell (like IPython), you can use the data.world Python package to run queries, import datasets, upload data, and more:

  1. Import the data.world package. You can always set a variable like dw for easier reference
    2. Without a variable: import datadotworld
    Using a variable: import datadotworld as dw
  2. Load a dataset using var = datadotworld.load_dataset and the dataset ID.
    3. Example: Note that if you've set a variable like dw in your import, you'll use that instead of datadotworld.
    intro_dw = dw.load_dataset('jonloyens/an-intro-to-dataworld-dataset')
  3. Start working with your data.

What next?

Here are some things you can do with Python and data.world:

Note that if you've set a variable like dw in your import, you'll always use that instead of datadotworld.
  • Use the Help function to get inline assistance on available functions.
    • help(dw)
  • Query a dataset using the var = datadotworld.query('dataset_ID', 'query') function.
    • query = dw.query('jonloyens/an-intro-to-dataworld-dataset', 'SELECT * FROM DataDotWorldBBallStats')
  • Use the data.world API client to complete a variety of functions, like uploading files to a dataset.
    • intro_dw.to_csv('dataset_intro.csv',index=False)
    client = dw.api_client()
    client.upload_files('jonloyens/an-intro-to-dataworld-dataset',files='dataset_intro.csv')
